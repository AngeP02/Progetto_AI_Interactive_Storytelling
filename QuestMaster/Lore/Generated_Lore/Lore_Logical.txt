# _Break_the_Ancient_Spell_

## 1. Mission Objective (Logical)
Collect the Golden Artifact to break an ancient spell.

## 2. World Structure

### Locations (Nodes):
- _start_
- _forest_path_
- _dark_cave_
- _ancient_castle_
- _treasure_vault_

### Location Connections (Edges):
- _start_ -> _forest_path_ (requires: none)
- _forest_path_ -> _dark_cave_ (requires: has_torch)
- _dark_cave_ -> _ancient_castle_ (requires: key_found AND guardian_defeated)
- _ancient_castle_ -> _treasure_vault_ (requires: puzzle_solved)

## 3. Items & Acquisition Logic

**torch**
  - Found at: _start_
  - Requires: none
  - Effect: can_enter_cave
**iron_key**
  - Found at: _forest_path_
  - Requires: has_torch
  - Effect: locked_gate_opened
**crystal_orb**
  - Found at: _dark_cave_
  - Requires: iron_key
  - Effect: puzzle_door_unlocked
**magic_sword**
  - Found at: _dark_cave_
  - Requires: crystal_orb
  - Effect: can_fight
**silver_key**
  - Found at: _forest_path_
  - Requires: has_torch
  - Effect: castle_door
**golden_artifact**
  - Found at: _ancient_castle_
  - Requires: magic_sword
  - Effect: quest_complete

## 4. Obstacles & Bypass Conditions

**locked_gate**
  - Blocks: _start_->_forest_path_
  - Requires to overcome: iron_key
  - Effect when solved: gate_opened
**cave_guardian**
  - Blocks: _forest_path_->_dark_cave_
  - Requires to overcome: magic_sword
  - Effect when solved: guardian_defeated
**puzzle_door**
  - Blocks: _dark_cave_->_ancient_castle_
  - Requires to overcome: crystal_orb
  - Effect when solved: puzzle_solved

## 5. Causal Dependency Chain

- BEFORE enter_castle: must have (silver_key AND guardian_defeated)
- BEFORE get_artifact: must have (puzzle_solved AND magic_sword)

## 6. Win Condition
```
at _treasure_vault_ AND has_golden_artifact AND quest_complete
```

## 7. PDDL Mapping

### Domain Actions:
- move(from, to) - Navigate between connected locations (requires: path conditions met)
- take(item, location) - Pick up an item (requires: at location, item available)
- use(item, target) - Use an item on obstacle/puzzle (requires: has item)
- solve(obstacle) - Overcome obstacle (requires: necessary items/conditions)
